# APEX Lite

Simple paper-trading bot with live dashboard.

## Features

- EMA crossover strategy
- Yahoo Finance market data
- Performance tracking
- Win rate tracking
- Balance history chart
- Flask dashboard

## Install

```bash
pip install -r requirements.txt
```

## Run Bot

```bash
python bot.py
```

## Run Dashboard

```bash
python dashboard.py
```

Open:

http://localhost:5000
